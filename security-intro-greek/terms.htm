Benchmarks
Ειδικά προγράμματα που εκτελούν μετρήσεις σε διάφορα υποσυστήματα ενός υπολογιστή και βγάζουν στατιστικά. Αυτά χρήσιμοποιούνται από περιοδικά του χώρου για να συγκρίνουν διάφορα προιόντα και να βγάλουν τα διάφορα γραφήματα για να πείσουν τον καταναλωτή ποιο προιόν είναι καλύτερο.

Cracks
Μικρά αρχεία ή προγράμματα που μετά την εγκατάσταση τους σας επιτρέπουν να παίξετε ένα παιχνίδι από ένα αντίγραφο και όχι από το αυθεντικό CD. Τα ίδια τα cracks καθώς και τα site που τα διαθέτουν είναι παράνομα.

DOS
Αρχικά που σημαίνουν Disk Operation System. To DOS ήταν ένα λειτουργικό σύστημα για προσωπικούς υπολογιστές φτιαγμένο από την Microsoft με μεγάλη διάδοση μέχρι και το 1995 (οπότε και βγήκαν τα Windows 95). Πολύ πρωτόγονο λειτουργικό σύστημα με τα σημερινά δεδομένα που γρήγορα ξεπεράστηκε.. Δεν είχε καν γραφικό περιβάλλον (μόνο γραμμη εντολών) ενώ ανάγκαζε τον υπολογιστή να τρέχει ένα πρόγραμμα την φορά (σε αντίθεση με σήμερα όπου ο υπολογιστής μπορεί να έχει πολλά προγράμματα φορτωμένα ταυτόχρονα.)

DSL
Αρχικά που σημαίνουν Digital Subscriber Line δηλαδή ψηφιακή συνδρομητική γραμμή. Ο όρος αναφέρεται σε σύνδεση με το διαδίκτυο που είναι μόνιμη και γρήγορη. Οι ταχύτητες που προσφέρονται είναι τάξεις μεγαλύτερες από τις απλές συνδέσεις 56k dialup.

Keylogger
Πρόγραμμα που καταγράφει όλα τα πλήκτρα που πατήθηκαν σε ένα υπολογιστή. Αυτό σημαίνει ότι μέσα σε αυτά θα είναι σίγουρα και όλοι οι κωδικοί (passwords) που χρησιμοποιήθηκαν όσο έτρεχε καθώς και άλλα προσωπικά δεδομένα (π.χ. πιστωτική κάρτα για αγορά μέσω διαδικτύου).

Shareware
Κατηγορία προγραμμάτων που διαθέτουν περιορισμένη λειτουργικότητα στην βασική τους έκδοση ενώ απαιτούν μικρό χρηματικό ποσό για την ενεργοποίηση όλων των δυνατοτήτων τους.

Patches
Η ελληνική μετάφραση είναι "μπαλώματα". Είναι οι διορθώσεις που κυκλοφορεί μια εταιρία μετά την κυκλοφορία του προιόντος της. Στην περίπτώση της Microsoft και των Windows XP τα patches πολλές φορές κλείνουν κενά ασφαλείας του λειτουργικού συστήματος οπότε καλό είναι τα εγκαταστήσετε. Αυτό γίνεται μέσα από το Windows Update.

Windows XP SP2
Η δεύτερη μεγάλη έκδοση των Windows XP με διάφορα μικρά χαρακτηριστίκά και πολλές διορθώσεις στον τομέα της ασφάλειας. Τα αρχικά σημαίνουν Service pack 2 (δεύτερο πακέτο διορθώσεων).

Windows 2000
Έκδοση των Windows που βγήκε το 1999 και απευθυνόταν σε επιχειρήσεις και επαγγελματίες παρά σε απλούς χρήστες. Η πρώτη "σοβαρή" εκδόση των Windows που δεν έπασχε από τα προβλήματα του παρελθόντος.
