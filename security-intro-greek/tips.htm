Εδώ θα βρείτε μερικές πρακτικές συμβουλές.

  • Αγνοήστε όλα τα περιοδικά που διαφημίζουν "τα 100 καλύτερα προγράμματα για την ασφάλεια του υπολογιστή σας". Μην εγκαταστήσετε τίποτα από το CD/DVD του περιοδικού. Σχεδόν όλα αυτά τα προγράμματα είναι άχρήστα. Ότι χρειάζεστε υπάρχει ήδη μέσα στο λειτουργικό σας σύστημα. Απλά πρέπει να ξέρετε που να το βρείτε.
  • Για να "δείτε" το firewall σας στα WindowsXP κάντε κλικ διαδοχικα Start menu->Connect to->Show all connetions. Στο παράθυρο που θα εμφανιστεί όσες συνδέσεις έχουν firewall θα έχουν ένα μικρό λουκέτο πάνω δεξιά στο εικονίδιο τους. Κάνοντας δεξί κλικ σε κάποιο εικονίδιο και επιλέγοντας properties μπορείτε μέσα από την καρτέλα advanced να διαλέξετε αν θέλετε να υπάρχει firewall ή όχι. Θα πρέπει να υπάρχει πολύ σοβαρός λόγος για να μήν θέλετε.
  • Αποφύγετε σελίδες του διαδικτύου μέ ύποπτο περιεχόμενο. Αυτές περιλαμβάνουν πορνογραφικό υλικό, παράνομα προγράμματα, cracks για παιχνίδια, καζίνα κτλ. Μην εγκαταστήσετε ότιδήποτε προσφέρεται σε αυτές τις σελίδες όσο δελεαστικό και αν είναι. Σε οποιαδήποτε ερώτηση εγκατάστασης απαντήστε ΟΧΙ στο παράθυρο που θα εμφανιστεί στην οθόνη σας.
  • Αγνοήστε επιδεικτικά όλα τα shareware /utilities/tweaks που προσφέρονται από περιοδικά και μεγάλα sites του χώρου. Θα πρέπει να ξέρετε επακριβώς τι κάνει ένα πρόγραμμα πριν το εγκαταστήσετε. Σε καμία περίπτωση μην υποκύψετε σε προγράμματα που διαφημίζουν επιτάχυνση του υπολογιστή σας. Αυτά είναι τα γνωστα RAM managers/internet boosters/accelerators κτλ.
  • Δώστε προσοχή σε όλα τα toolbars/launchers/clock/desktop apps και συναφή που διαφημίζονται. Τα περισσότερα προγράμματα από αυτά είναι spyware. Το ίδιο ισχύει και για τους download managers.
  • Μην ακούτε φίλους και γνωστούς που το παίζουν ειδικοί και σας λένε ότι αν βάλετε το τάδε πρόγραμμα θα είστε ασφαλείς. Πείτε τους οτί μπορούν να κάνουν ότι θέλουν στον δικό τους υπολογιστή, αλλά τον δικό σας τον διαχειρίζεστε μόνο εσείς.
  • Αντιμετωπίζετε όλα τα site που επισκέπτεστε στο διαδίκτυο με δυσπιστία. Προσέξτε αυτά που σας ζητάνε να γραφτείτε (registration). Καλό θα ήταν να μην δίνετε πάντα όλα τα προσωπικά σας στοιχεία.
  • Μάθετε να ξεχωρίζετε ποια παράθυρα είναι διαφημίσεις για προίοντα και ποια είναι πραγματικά μηνύματα του υπολογιστή σας. Προσέξτε που κάνετε κλικ. Αγνοήστε όλα τα site που λένε ότι ο υπολογιστής δεν είναι ασφαλής/έχει πρόβλημα/είναι αργός και άλλα τέτοια (και σας προσφέρουν και καλά κάποια λύση).
  • Για να ενεργοποίησετε το Windows Update που θα κατεβάσει για εσάς όλες τις τελευταίες ενημερώσεις (Patches) από το διαδίκτυο και θα τις εγκαταστήσει στον υπολογιστή σας, ανοίξτε ένα οποιόδηποτε παράθυρο του Internet Explorer και από το μενού Tools κάντε κλικ στην επιλογή Windows Update. Εναλλακτικά μπορείτε να το ξεκινήσετε και από το Start menu ή από τον πίνακα ελέγχου.
  • Για να σβήσετε όλα τα cookies του Internet Explorer κάντε κλικ στο μενού Tools όπως προηγουμένως και μετά στην επιλογή Internet Options. Στο παράθυρο που θα εμφανιστεί κάντε κλικ στο κουμπί Delete Cookies... και πατήστε OK για επιβεβαίωση. Μπορείτε να κλείσετε το παράθυρο των επιλογών μετά.
